Flagship Case Study

Employee Attrition Analytics Dashboard

SQL (MySQL) Power BI Tableau Python Pandas & Scikit-Learn

01 // Objective & Problem Scope

High employee turnover carries heavy organizational costs, including lost institutional knowledge, recruitment overhead, and onboarding friction. This project maps an end-to-end data pipeline to analyze historical HR cohorts and diagnose operational drivers of attrition.

By structuring synthetic employee data from disconnected spreadsheets into a normalized relational model, I built diagnostic querying suites and interactive BI dashboards to help stakeholders isolate at-risk segments and make data-backed retention decisions.

Business Impact

  • • Identify high-risk roles & divisions
  • • Measure overtime burnout correlations
  • • Benchmark salary brackets & retention
  • • Propose actionable HR policies

02 // Technical Architecture & Data Flow

The end-to-end data pipeline processes disjointed Excel datasets into a structured SQL database schema, followed by custom diagnostic queries and machine learning churn models, ultimately serving interactive reports to business decision makers.

Figure 1. High-resolution system schematic detailing raw data ingestion, relational schema normalization, insights querying, and visualization layers.

03 // Database Schema & Relational Normalization

The raw employee spreadsheets were separated into two core logical tables inside a relational database to enforce data integrity and enable clean query joins:

Table 1. Demographics (hr_1)

Stores immutable variable cohorts: `EmployeeNumber` (PK), `Age`, `Gender`, `MaritalStatus`, `EducationField`, and qualitative job satisfaction surveys.

Table 2. Job & Compensation (hr_2)

Stores operational variables linked via `EmployeeID` (FK): `MonthlyIncome`, `PercentSalaryHike`, `Overtime` check, `YearsAtCompany`, and `YearsSinceLastPromotion`.

04 // SQL Diagnostic Query Showcase

These SQL queries isolate key indicators of attrition, demonstrating conditional groupings, data aggregation, and table joins.

Cohort Analysis A // Attrition Rate by Income Bracket 

SELECT 
    CASE 
        WHEN t2.monthlyincome < 3000 THEN 'Under $3k'
        WHEN t2.monthlyincome BETWEEN 3000 AND 6000 THEN '$3k - $6k'
        WHEN t2.monthlyincome BETWEEN 6001 AND 10000 THEN '$6k - $10k'
        WHEN t2.monthlyincome BETWEEN 10001 AND 15000 THEN '$10k - $15k'
        ELSE 'Over $15k'
    END AS income_bracket,
    COUNT(*) AS total_employees,
    SUM(CASE WHEN t1.attrition = 'Yes' THEN 1 ELSE 0 END) AS attrition_count,
    ROUND(SUM(CASE WHEN t1.attrition = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS attrition_rate_pct
FROM hr_1 t1
JOIN hr_2 t2 ON t1.employeenumber = t2.employeeid
GROUP BY income_bracket
ORDER BY MIN(t2.monthlyincome) ASC;

Cohort Analysis B // Burnout Metrics (Overtime & Low Raises) 

SELECT 
    t2.overtime,
    CASE WHEN t2.percentsalaryhike < 15 THEN 'Low Hike (<15%)' ELSE 'Normal/High Hike (15%+)' END AS hike_status,
    COUNT(*) AS total_employees,
    SUM(CASE WHEN t1.attrition = 'Yes' THEN 1 ELSE 0 END) AS attrition_count,
    ROUND(SUM(CASE WHEN t1.attrition = 'Yes' THEN 1 ELSE 0 END) * 100.0 / COUNT(*), 2) AS attrition_rate_pct
FROM hr_1 t1
JOIN hr_2 t2 ON t1.employeenumber = t2.employeeid
GROUP BY t2.overtime, hike_status
ORDER BY attrition_rate_pct DESC;

06 // Strategic HR Recommendations

Data-Backed Insights:

  • Overtime Impact: Employees working overtime experience double the attrition rates (over 30% exit risk).
  • Income Floors: Retention issues are highly concentrated in the sub-$3,000 monthly income bracket.
  • Career Stagnation: Employees who pass the 12-month tenure point without an active promotion show a high tendency to churn.
