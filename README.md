# Vardaan Bajaj — Engineering Portfolio & Manuscripts

A premium, restrained atmospheric portfolio site showcasing applied machine learning research, data science dashboards, and academic publications. Designed around a dark walnut, gold, forest green, and copper palette with delicate typographic pairings.

---

## 🏛️ Flagship Case Studies

### [Employee Attrition Analytics Dashboard](employee-attrition.html)
An end-to-end data analytics system mapping corporate flight risk. Normalizes messy telemetry data into a structured MySQL database, implements custom cohort diagnostics, and serves interactive dashboards.
- **Technologies:** SQL, Python, MySQL, Power BI, Tableau, Random Forest ML, ETL

### [NeuroSight AI: Parkinson's Prediction](neurosight-ai.html)
An explainable machine learning model exploring vocal dysphonias for early-stage Parkinson's Disease detection. Employs a custom Feature Performance Index (fPI) to identify top acoustic features.
- **Technologies:** Python, XGBoost, Scikit-Learn, Pandas, EDA

## 📄 Academic Contributions & Publications

### [OOP-Based Web Page Linker](web-page-linker.html) — *IEEE IATMSI*
An object-oriented parsing module that extracts DOM functional divisions (`<div>`) and encapsulates them into reusable, polymorphic Python class modules to bypass DOM fragility.
- **Technologies:** Python, Requests, Beautiful Soup, OOP Design Patterns

### [V-Surveillance Drone Framework](v-surveillance.html) — *IEEE CICT*
A three-stage real-time drone imagery deep learning pipeline integrating Super Resolution, object detection, and tracking for edge-computing UAV surveillance.
- **Technologies:** ESRGAN, YOLO12M + SAHI, Deep SORT, PyTorch, UAV Edge Nodes

## 🛠️ Development & Building

This portfolio is built using **React**, **Vite**, and **Tailwind CSS**.

### Setup
Install dependencies:
```bash
npm install
```

### Run Locally
Start the development server:
```bash
npm run dev
```

### Build for Production
Vite compiles and outputs the production bundle under the `/dist` directory, resolving all multi-page entry points:
```bash
npm run build
```
